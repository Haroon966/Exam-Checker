# Pinky + Claude

This project uses the Pinky multi-agent kit. Follow [AGENTS.md](AGENTS.md) as the source of truth.

**Summary:** Ponytail ladder before you write code. Caveman terse replies (code/errors exact). Small diffs, no secrets, one check for non-trivial logic.

**Before a large feature:** run `/grill-with-docs` (or `/grill-me` if you do not want files written).

**UI work:** use `frontend-craft` and `/ui-ux-pro-max` when available. **API/server work:** use `backend-apis`.

Skills live under `.claude/skills/` (and mirrored host paths). Customize via [docs/pinky-customization-guide.md](docs/pinky-customization-guide.md).
